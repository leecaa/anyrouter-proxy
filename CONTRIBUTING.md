# Contributing

We welcome contributions to this project! Here's how you can help:

1. **Fork the repository** and create your branch from `master` or `main`.
2. **If you've added code that should be tested, add tests.**
3. **Ensure the test suite passes.**
4. **Make sure your code lints.**
5. **Issue a pull request.**

## Guidelines
- Do not commit secrets, API keys, or personal configuration files.
- Ensure that any new placeholder keys are strictly obvious placeholders (e.g., `sk-ant-xxx`).
- Avoid pushing real IP addresses to the `指纹/` (fingerprint) sample configurations.
